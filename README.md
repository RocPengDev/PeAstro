# 🚀 PeAstro
使用Astro框架，自动化部署到Cloudflare的静态博客项目，不需要本地部署，不需要额外购买服务器。

### 特色
- 使用GitHub Issues 作为博客内容编辑工具
- 将issues转换为博客内容
- 自动化内容同步和部署
- 针对Cloudflare部署优化
- 静态站点生成，实现最优性能
